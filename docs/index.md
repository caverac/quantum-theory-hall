# Solutions to the problem sets from Brian C. Hall's Quantum Theory for Mathematicians

!!! info "Version"
    Current version: **v{{ version }}**

This repository contains solutions to various problem sets from Brian C. Hall's *Quantum Theory for Mathematicians*. The code is organized into modules that correspond to different chapters and topics covered in the book.

Numerical solutions are implemented in Python, leveraging libraries such as NumPy, SciPy, and Matplotlib for efficient computation and visualization.
